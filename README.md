# Portfolio Website

A clean, modern portfolio website showcasing visual stories and interactive projects.

## Features

- Clean, minimalist design
- Carousel component for visual stories
- Responsive layout for all devices
- Modern CSS with variables for easy customization

## Structure

- `index.html` - Main page
- `css/styles.css` - Main stylesheet
- `js/carousel.js` - Carousel functionality

## Setup

1. Clone the repository
2. Customize the content in `index.html`
3. Add your own projects and images
4. Deploy to your hosting provider

## Image Directory Structure

The site expects images in the following directories:

```
images/
├── stories/ (for carousel images)
```

## Customization

You can easily customize the site by:
- Editing CSS variables in the `:root` selector
- Adding more carousel items in the HTML
- Changing the layout in CSS
